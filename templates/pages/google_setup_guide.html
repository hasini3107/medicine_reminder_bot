Configure Google Sign-In

Please configure your Google OAuth Client ID in config.py to redirect users to Google's official login screen.

📋 Setup Steps

  • Go to the Google Cloud Console.
  • Navigate to APIs & Services > Credentials.
  • Click Create Credentials > OAuth client ID.
  • Set Application Type to Web application.
  • Add the following URL to Authorized redirect URIs:
    http://127.0.0.1:5000/api/auth/google/callback
  • Copy the Client ID and Secret and paste them into your config.py.
